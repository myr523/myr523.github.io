## develop
- setup
```shell
$ git@github.com:myr523/myr523.github.io.git
$ cd myr523.github.io
$ npm i
```

- test
```
$ npm run test
```

- start local-dev-http-server
```
$ npm start
```

- build
```
$ npm run build
```
